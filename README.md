# Suica SF利用履歴管理 PWA

## 概要
モバイルSuicaの「SF利用履歴」スクリーンショットをOCR（文字認識）で自動解析し、  
個人の交通費・支出をiPhoneで手軽に把握・管理できるWebアプリ。  
Vercelにデプロイ済み。iPhoneのSafariからホーム画面に追加してアプリとして使用可能。

## ファイル構成
```
suica-pwa/
├── index.html   ← アプリ本体（HTML / CSS / JavaScript すべて1ファイル完結）
└── README.md    ← この説明書
```

## 機能一覧

### 画像の読み込み
| 方法 | 説明 |
|------|------|
| 📷 ファイル選択 | ボタンをタップして画像を選ぶ（複数枚同時対応） |
| 📋 貼り付け | iPhoneでコピーしたスクリーンショットを直接貼り付け（HTTPS環境のみ） |

### OCR解析
- Tesseract.js（日本語＋英語）で金額・日付・利用場所を自動抽出
- 画像をグレースケール化＋コントラスト強化してから解析（精度向上）
- 入場駅・出場駅を「戸越→都人形町」形式で表示
- 同じ内容（日付＋金額＋場所）は重複登録しない

### 集計機能
- **25日始まりの集計期間**（例: 3/25 〜 4/24）で管理
- ◀▶ ボタンで過去の月も遡って確認できる
- 各カードのスイッチで集計ON/OFFを切り替え
- ヘッダーに「集計ONの合計」を大きく表示
- **合計金額を長押しするとクリップボードにコピー**（数字のみ）

### カード操作
- メモ欄に「通勤」「ランチ」などを自由入力（自動保存）
- 🗑 ボタンで個別削除（確認ダイアログあり）
- 「全データをリセット」で一括削除

### データ管理
- データはすべてブラウザの localStorage に保存（サーバー送信なし）
- アプリを閉じても消えない・蓄積されていく
- localStorage 上限: 約5MB（数万件分。通常使用では問題なし）

## iPhoneでの使い方

### 初回セットアップ
1. iPhoneのSafariでVercelのURLを開く
2. 共有ボタン →「ホーム画面に追加」→「追加」
3. ホーム画面のアイコンからアプリとして起動

### スクリーンショットの追加（2通り）

**方法A: ファイル選択**
1. 「📷 スクリーンショットを追加」をタップ
2. 写真ライブラリからSuica履歴の画像を選ぶ

**方法B: コピーして貼り付け**
1. 写真アプリでスクリーンショットを開く
2. 共有ボタン →「コピー」
3. アプリに戻り「📋 コピーから貼り付け」をタップ
4. 「貼り付けを許可しますか？」→「許可」

## 技術仕様

| 項目 | 内容 |
|------|------|
| OCR | Tesseract.js v5（CDN）、日本語+英語モード |
| データ保存 | localStorage のみ（端末内・サーバー不使用） |
| 集計期間 | 毎月25日始まり（例: 3/25〜4/24） |
| 対応環境 | iPhone Safari / Chrome（PC） |
| アクセントカラー | Suicaグリーン #00B900 |
| デプロイ | Vercel（無料プラン） |

---

## 注意事項
- 初回起動時、Tesseractの日本語データ（約7MB）をダウンロード（インターネット接続が必要）
- 「📋 貼り付け」機能はHTTPS環境（Vercel等）でのみ動作。ローカルの `index.html` では使用不可
- OCRの読み取り精度はスクリーンショットの画質に依存。鮮明な画像ほど精度が上がる
- データはブラウザ・端末ごとに別管理。機種変更時はデータが引き継がれない
